‚öôÔ∏è Installation

üìã Prerequisites

‚ö†Ô∏è Requirements:
  • Node.js (version 18.0 or higher)
  • npm or yarn package manager
  • Git (for cloning the repository)
  • Modern web browser (Chrome, Firefox, Safari, Edge)

üîΩ Step-by-Step Installation

1 Clone the Repository
git clone https://github.com/yourusername/chat-lab.git cd chat-lab
2 Install Dependencies
npm install # or if you prefer yarn yarn install
3 Environment Setup (Optional)

Create a .env file in the root directory if you need custom configurations:

# .env file VITE_API_BASE_URL=http://localhost:3000 VITE_APP_TITLE=FIDU Chat Lab - AI Conversation Manager
4 Verify Installation
npm run dev

If successful, you should see output similar to:

VITE v6.3.5 ready in 249 ms ‚ûú Local: http://localhost:5173/ ‚ûú Network: use --host to expose
‚úÖ Installation Complete! Your FIDU Chat Lab application is now installed and ready to use.

üîß Troubleshooting

‚ùå Common Issues

1 Port Already in Use
Problem: "Port 5173 is in use"
Solution: Vite will automatically find the next available port. Check the terminal output for the actual URL.
2 Dependencies Issues
Problem: npm install errors
Solution:
# Clear npm cache npm cache clean --force # Delete node_modules and reinstall rm -rf node_modules package-lock.json npm install
3 Browser Compatibility
Problem: Page not loading correctly
Solution:
  • Use a modern browser (Chrome 90+, Firefox 88+, Safari 14+)
  • Clear browser cache and cookies
  • Disable browser extensions temporarily
  • Check browser console for errors (F12)

üîç Debug Mode

1 Enable Debug Logging

Add this to your .env file:

VITE_DEBUG=true VITE_LOG_LEVEL=debug
2 Check Browser Console

Press F12 to open developer tools and check the console for errors.
