ACM Manager - Update Functionality Test

Test Overview

This page tests the new update functionality for ACMs. The system now:

  • Catches 409 CONFLICT errors when trying to create an ACM that already exists
  • Performs a PUT update to replace the existing data entirely
  • Uses the current view of the conversation as the authoritative version
  • No longer merges data to avoid duplication issues
